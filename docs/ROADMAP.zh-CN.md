# 路线图

**语言：** [English](ROADMAP.md) | 中文

本路线图描述 `familycut` 公开仓库的当前状态，并区分已经可用的内容和每个使用者需要在自己环境里补完的内容。

## 已经足够使用

- 后端领域模型和 API
- 管理端和移动端 Web 骨架
- Docker 后端栈
- 可公开的架构说明

## 需要本地补全

- 首个家庭初始化/引导流程
- 生产级鉴权和会话策略
- 个人数据备份和恢复流程
- Android release 构建流程

## 建议改进

- 增加首次运行设置向导
- 增加成员和测量流程端到端测试
- 改进移动端 PWA 离线行为
- 补充备份/恢复操作文档

## 仍值得补充的文档

- 使用非私有演示数据制作截图或短录屏。
- 为常见请求/响应补完整 API 示例页。
- 为持久化数据路径补备份和恢复说明。
- 等公开用户开始部署后，基于真实问题整理排障页。

## 维护说明

- 公开示例保持通用。
- 英文和中文说明保持同步。
- 尽量拆小 issue 和 pull request，方便 AI 辅助贡献者安全接手。
- 发布新版本前重新跑敏感信息扫描。
