# Screenshots and Demo Media

**Language:** English | [中文](SCREENSHOTS.zh-CN.md)

This public repository does not commit private deployment screenshots. Put public-safe screenshots in `docs/screenshots/`, using only synthetic data, public demo data, or redacted content.

## Recommended Screenshots

- Admin dashboard using seeded demo household data
- Mobile daily input flow using demo data
- Reports page with synthetic measurements only

## Pre-Publish Checklist

- No real names, chat logs, media titles, health records, order details, cookies, tokens, local network addresses, or production URLs.
- Use clear filenames such as `overview.png`, `settings.png`, or `report-demo.png`.
- Verify path casing before linking images from `README.md` or this file.
- Update the Chinese `docs/SCREENSHOTS.zh-CN.md` when screenshot guidance changes.

## Suggested Directory

```text
docs/screenshots/
  overview.png
  settings.png
  workflow.png
```
